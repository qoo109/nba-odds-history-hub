## Public files

The snapshot is split into a small index plus one JSON file per market under `data/public/snapshots/2026-07-20/`. The browser can generate full JSON and CSV downloads without committing a large database.

`data/public/manifest.json` records file hashes, counts, matching results, quality flags, and the public usage boundary.

## Interpretation boundary

`capturedAt` is the observation time. `cutoffAt` is the market deadline and must not be substituted for `capturedAt`.

The dashboard displays the sum of raw implied probabilities as a descriptive overround indicator. It is not a no-vig model.

Only one observation time exists, so historical movement, opening/closing classification, CLV, EV, ROI, and NBA Value Lab market backtesting remain blocked.
